# BT-Structure VR Tour

Proyecto WebVR creado en A-Frame para presentar un recorrido visual y narrado sobre la teoría BT-Structure.

## Archivos incluidos

- `index.html` — estructura principal y escena VR
- `scene.js` — controles del tour y animaciones
- `style.css` — diseño del menú inicial
- `narration.ogg` — narración del recorrido (no incluida)
- `README.md` — documentación

## Cómo usar

1. Coloca todos los archivos en la misma carpeta.
2. Asegúrate de incluir `narration.ogg` (audio en formato OGG).
3. Sube el proyecto a GitHub o ábrelo desde un servidor local.
4. Entra en la URL y pulsa **"Iniciar tour"**.

El tour funciona tanto en PC, móvil como en gafas VR (Meta, Pico o equivalentes).
